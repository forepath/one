# Knowledge graph artifacts

Generated by `@forepath/graph`:

```bash
nx run graph:generate-kg
```

| File         | Description                                                      |
| ------------ | ---------------------------------------------------------------- |
| `graph.json` | Unified knowledge graph (projects, files, APIs, concepts, edges) |
| `graph.html` | Force-directed graph viewer (Obsidian-style) + detail panel      |

View the HTML over HTTP (required because it fetches `graph.json`):

```bash
nx run graph:serve
```

Then open http://127.0.0.1:4211/

These files are **committed**. The Husky pre-commit hook regenerates them and runs `git add` so each commit includes an up-to-date graph without further intervention.
